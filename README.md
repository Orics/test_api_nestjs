## Description

A template for API using Nest framework TypeScript.

## Installation

```bash
# install packages
$ npm install

# install global packages
$ npm install -g @nestjs/cli nodemon pg ts-node typeorm typescript
```

## Running the app

```bash
# run
$ npm run start

# log
$ npm run log

# stop
$ npm run stop
```
